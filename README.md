#Overview
This project focuses on analyzing and benchmarking the performance of Node.js using Worker Threads, Promises, and Performance Hooks.
It performs word frequency analysis on a large text dataset (corpus.txt) generated using the Lorem Ipsum library, while measuring execution time under different concurrency levels 1, 4, 8).
In this we will compute:
> Total words
> Unique words
> Longest and shortest words
> Top N most repeated words
